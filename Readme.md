❓ Quiz App
An interactive quiz application where users can answer multiple-choice questions and receive a final score at the end.


🚀 Live Demo
https://atalan04.github.io/Quiz-App/


🛠 Technologies Used
HTML5
CSS3
JavaScript (ES6)


✨ Features
Multiple-choice questions
Instant answer selection
Score calculation
Final result display
Responsive layout


🧠 What I Learned
Managing application state in JavaScript
Working with arrays and objects for quiz questions
Updating the UI dynamically with DOM manipulation
Handling user interaction and event listeners


⚙️ How to Run Locally
git clone https://github.com/Atalan04/Quiz-App.git


📌 Future Improvements
Timer for each question
More quiz categories
